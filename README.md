# BreakoutJack
This is an implementation of the breakout game in "Jack" (nand2tetris)

##HowTo
* Load into the VM Emulator
* Choose No Animation
* Run Game

##Controlls
* [LEFT-ARROW]: Move bat left
* [RIGHT-ARROW]: Move bat right
* [+]: Skip the level (Not officially supported)

##Finally...
...Have fun!
